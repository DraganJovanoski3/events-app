Overview

This project is an Angular application that allows users to view events based on their location and register as users. It uses geolocation to fetch events relevant to the user's city.

Features

Event List

Displays a list of events based on the user's location.

Login

Allows users to log in to their accounts.

Register

Provides a form for new users to register with validation.
